Python is a programming language that has been rapidly growing in popularity in the last few years. It is being utilized in many different fields, such as AI, data analysis and web development. By taking this course you will learn the fundamentals of Python and its base concepts. Starting from simple operators, to writing functions, classes, code blocks and finally, building your own website. This course is suitable for beginners, no prior knowledge of programming is required. If you are interested in programming but don’t know where to start, this is the course for you. Python is remarkably simple compared to many other languages and provides a good foundation for you to start building your knowledge upon.

    After finishing this course, you will know:

  • Fundamentals of Python Programming;
  • How to write basic and effective code;
  • How to store, modify and send data;
  • How to build a simple website.
